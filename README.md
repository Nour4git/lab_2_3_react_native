# Notes App - React Native

# Overview

A secure note-taking application with user authentication for personalized notes, developed with React Natuve and Appwrite.

# Features

- User authentication
- CRUD operations for notes (Create, Read, Update, Delete)
- User-specific data isolation
- Simple and modern interface
- Cross-platform support (Android, iOS)

# Technologies

- **Recat Natve** - UI Framework  
- **Appwrite** - Backend  
- **Java Script** - Programming Language

# Setup

1. `git clone https://github.com/Nour4git/lab2_3_react_native.git`
2. `cd NotesApp`

# .env File Configuration

```env
APPWRITE_ENDPOINT=https://cloud.appwrite.io/v1
APPWRITE_PROJECT_ID=your-project-id
APPWRITE_DATABASE_ID=notes_db
APPWRITE_COLLECTION_ID=notes
```

# Run 
npx Expo Start
